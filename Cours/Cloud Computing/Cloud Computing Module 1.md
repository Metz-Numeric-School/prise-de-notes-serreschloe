**Cloud computing** : C'est le déploiement **à la demande** de puissance de calcul, de stockage, de bases de données, d'applications et d'autres ressources informatiques **par internet, avec tarification à l'utilisation**

## Avantages du Cloud

- Transformer les dépenses d'investissement en coûts variables 
- Economies d'échelle importante
- Plus besoin de deviner la capacité nécessaire
- Augmenter la vitesse et l'agilité
- Ne dépensez plus d'argent pour le fonctionnement et la maintenance des centres de données
- Déploiement international en quelques minutes

## Introduction à AWS

### Que sont les services web ?

Un service web est un logiciel qui se rend disponible sur Internet et utilise un format normalisé, tel que XML ou JSON pour la demande et la réponse d'une interaction d'API

### Trois moyens d'interagir avec AWS

- AWS Management Console
- Interface de ligne de commande (AWS CLI)
- Kits de développement logiciel (SDK) -> pour les DEVS
